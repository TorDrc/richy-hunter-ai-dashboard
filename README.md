 # 🚀 Richy Hunter AI
## Solana Memecoin Scanner
Richy Hunter AI est un dashboard mobile destiné à analyser rapidement les nouveaux tokens Solana.
## Fonctionnalités V1
✅ Interface iPhone  
✅ Analyse d'un token DexScreener  
✅ Score Hunter AI /100  
✅ Analyse risque  
✅ Signal :
- 🟢 Entry
- 🟡 Watch
- 🔴 Avoid

- Prix du token

- Achats/Ventes
- Variation %
- Analyse automatique
## Installation
1. Copier les fichiers dans GitHub Pages :

index.html
style.css
app.js

2. Activer GitHub Pages.
3. Ouvrir le lien du site sur iPhone.
## Utilisation
1. Copier un lien DexScreener Solana
Exemple :

https://dexscreener.com/solana/ADRESSE_TOKEN

2. Coller dans Richy Hunter AI
3. Cliquer :
SCAN TOKEN
